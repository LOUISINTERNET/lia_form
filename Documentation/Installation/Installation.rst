============
Installation
============

..  _installation-composer:

Install LIA Form with Composer
=======================================

Install the extension via Composer:

..  code-block:: bash

    composer req lia/lia_form

See also `Installing extensions, TYPO3 Getting started <https://docs.typo3.org/permalink/t3start:installing-extensions>`_.

..  _installation-classic:

Install LIA Form in Classic Mode
=========================================

Or download the extension from `https://extensions.typo3.org/package/lia_form <https://extensions.typo3.org/package/lia_form>`_ and install it in
the Extension Manager.
